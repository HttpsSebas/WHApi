#WHApi messaging

## Stack
- NodeJS
- Express
- Bcrypt
- Dotenv
- Prisma
- WebSockets (socket.io)
- PostgreSQL
- JWT + Refresh Token
- Zod
- Redis
- Docker

## Features
- JWT Authentication + Refresh
- Separation by layers (controllers, services, events, etc)
- Real-time chats (group & private)
- Centralized error handling
- Auth Middleware
- Instantly messaging with Socket.io
- Sockets automatic reconnection
- Message persistence
- Multi-device supported
- Message sync
- Room system for chats
- Online/Offline user presence
- Active sessions handling
